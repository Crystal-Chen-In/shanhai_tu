# 山海途

> 以《山海经》神兽白泽为伴，融合任务管理与专注计时的修行系效率工具。

## 核心功能

- **任务管理**：创建、编辑、删除任务；设定截止日期；标记重要程度；滑动删除；按截止日期排序。
- **专注计时**：自定义专注时长（可调）；圆形进度条；开始/暂停/重置；完成后增加修为值。
- **白泽智能反馈**：根据任务完成情况（提前/按时/逾期）、连续修行天数、任务完成率、重要任务、专注时长等规则，弹出个性化鼓励台词（带淡入动画与呼吸效果）。
- **修行日记**：展示连续修行天数、累计专注时长、任务完成率，帮助用户直观看到成长。

## 技术栈

- Flutter (Dart)
- shared_preferences（本地数据持久化）
- Timer（计时器）
- AnimationController（弹窗动画）
- FutureBuilder（异步数据加载）
- jsonEncode / jsonDecode（任务序列化）
- Material Design（UI 组件）

## 如何运行

1. 安装 Flutter 环境（[官方指南](https://flutter.cn/docs/get-started/install)）
2. 克隆本项目或解压源码包
3. 在项目根目录执行：
   ```bash
   flutter pub get
   flutter run
首次运行会自动下载依赖，请确保设备已连接或模拟器已开启。

## 注意事项

- 首次安装后，数据均保存在本地，卸载应用会清除所有记录。
- 专注时长可手动调整，但当前版本不会自动保存用户偏好（每次打开默认 25 分钟）。
